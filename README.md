# Uzair's Portfolio Website 🚀

A modern, responsive portfolio website built with pure HTML, CSS, and JavaScript.

## 🌐 Live Demo
[View Portfolio](https://uziikhan.github.io/uzii-portfolio)

## 📸 Preview
![Portfolio Preview](preview.png)

## ✨ Features
- Responsive design (mobile friendly)
- Smooth scroll animations
- Custom cursor effect
- Animated skill bars
- Project showcase with live links
- Contact section with Gmail and Fiverr links
- Dark modern theme

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📂 Project Structure
```
uzii-portfolio/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🚀 Getting Started
1. Clone the repository
```bash
git clone https://github.com/uziikhan/uzii-portfolio.git
```
2. Open `index.html` in your browser
3. Done!
